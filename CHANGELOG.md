# Changelog

All notable changes to this project are documented in this file.

## [v0.2-demo] - 2025-11-03
### Added
- Subtle demo footer in the sidebar with link to David X Avila's portfolio.
- Notebook viewer (frontend) with markdown rendering and syntax highlighting; backend notebook model and route.
- Responsive audit tooling and example screenshots used to validate layout across device sizes.

### Changed
- Header search: disabled autofill, added honeypot inputs, improved accessibility.
- Papers page: differentiated page-level search UI and removed redundant New Paper button.
- Data Hub: unified page heading style and replaced upload link with a styled button.
- Dashboard & Papers: demo dates are now dynamically computed so the demo always appears up-to-date.

### Misc
- Small CSS, layout polish, and other demo-focused improvements.
